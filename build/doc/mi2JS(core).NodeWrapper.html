Class: NodeWrapper

mi2JS. NodeWrapper

new NodeWrapper()

Very basic wrapper used for scripting(similar to JQuery and alike). Contains some useful methods to allow manipulation, and can be extended further if needed. For more advanced functionalities check mi2JS.comp..Base component and it's descendants in mi2JS.comp..

Methods

addClass()

Add a css class to the element. Common function to initiate change defined in css.

attr(name, value)

get/set attribute on the wrapped node. Setting attribute to null or undefined will remove the attribute.
Parameters:
Name Type Description
name String attribute name
value String optionaly if sent sets the attribute

attrBoolean(name, def)

Boolean from attribute
Parameters:
Name Type Description
name string attribute name
def object default value if attribute is not present
Example
Value is true for all these instances
just defined - <input required> <input required="">
value same as name <input required="required">
value 1,true - <input required="1"> <input required="true">

attrDef(name, def)

Get attribute, but return default value if not defined
Parameters:
Name Type Description
name string attribute name
def object default value if attribute is not present

attrNum(name, def)

Get attribute, but convert frist using mi2JS(core).num. Returns 0 when value not a number or if attribute is not present.
Parameters:
Name Type Description
name string attribute name
def object default value if attribute is not present

classIf(toAdd, condition)

Add class to the element if condition is true, and remove if false.
Parameters:
Name Type Description
toAdd className to add/remove
condition (true/false) determines if add/remove is executed. Usualy a result of an expression in the caller code.
