Type Definitions

ElementTemplate

Defines properties needed to extract single element template from a a node, or to programatically define such template.
Type:
  • object
Properties:
Name Type Description
tag string tag name
attr Object object containing key->value pairs for attributes
html string innerHTML of the element

EventObject

Event object used for firing events inside components

src and target are added automatically, you can not pass them to fireEvent

If you listen to 'submit' event on a Form, src will be the input that originated the 'submit' event and target will be the Form

Type:
  • object
Properties:
Name Type Description
name string event name
fireTo string direction of the event (undefined|parent|children)
src Object source/origin component where of the event
target Object target component the event is firing from right now
domEvent Event (optional) should be passed when DOM Event was the cause

TagTemplate

Tempalte object for passing info about a HTML node taht should be created

Used by mi2JS(html).addTag, mi2JS(html).add

Type:
  • object
Properties:
Name Type Description
tag string tag name (uppercase)
html string html code to put into innerHTML of the newly created node
attribs Object name/value pairs for desired attributes and the values
