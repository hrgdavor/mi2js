mi2JS html

Methods

<static> add()

Create a html node, and returns a mi2JS(core).NodeWrapper. Uses mi2JS(html).addTag with same parameters
but returns the created node wrapped with mi2JS(core).NodeWrapper.
See:
  • mi2JS(html).addTag

<static> addTag(parent, tpl, tpl)

Create a html node.
Parameters:
Name Type Description
parent Element parent for the new node. Use null if you intend to add the node to a parent later using appendChild(..) .
tpl string | TagTemplate tag name, or tag template object must be uppercase
tpl Object tag name, must be uppercase
Returns:
new node
Type
Element
Example
mi2.addTag(document.body, 
		{	tag:'DIV', 
			html: '<b>Title:</b>Sunshine', 
			attr:{'class':'freaky-line'}
		}
	);
	// returns creaed Element
